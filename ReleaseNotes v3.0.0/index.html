Version 3.0.0

Release Notes

06.02.2024

With the release of version 3, Springboard now includes a translation feature! Users can now translate articles from/into more than 130 languages.

🚀New Features
  • Translation. Users can now translate articles into more than 130 languages.
⚡Enhancements
  • Menus have been updated
  • Improved customization process. Improvements include an help screen prior to selecting fonts, more clear and concise prompts, and more!
⚠️Known Issues
  • When performing multiple actions in a row from the Settings menu, each action results in a "Run Shortcut" action to take the user back to the Settings menu. However, if last option selected is "Exit", the shortcut does not stop but rather prompts the user to select how to view an article despite the shortcut not having any input (article).
